# WANT TO CREATE AND EDIT FILES SIMULTEANOUSLY, THEN THESE COMMANDS ARE FOR YOU : 

nano <filename.txt><br>
vi <filename.txt> <br>
vim <filename.txt> <br>
neovim <filename.txt> <br>
gedit <filename.txt><br>
kate <filename.txt><br>


<h4>These tools come with the Kali default installation and opens the file after creation. Contents specified in the file are not saved, until the save command is executed. Tools
such as 'gedit' and 'kate' are to be installed before command use. </h4>
