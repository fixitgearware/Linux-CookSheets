# TO CREATE FILES VIA CLI USING COMMAND ONLY USE THE COMMAND BELOW:

touch <filename.txt><br>
touch <filename1.txt> <filename2.txt> <filename3.txt><br>
touch -c filename.txt <br>
> filename.txt ----> to create or truncate file <br>
>> filename.txt ---> to create or append to file <br>
: > filename.txt -->  to create an empty file <br>
